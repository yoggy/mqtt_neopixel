How to use:
----

    $ git clone https://github.com/yoggy/mqtt_neopixel
    $ cd mqtt_neopixel
    $ cp config.ino.sample config.ino
    $ vi config.ino
      - edit wifi_ssid, wifi_password, mqtt_server, mqtt_subscribe_topic, ... etc
    $ open mqtt_neopixel.ino
